# 🚖 Power BI Ride Booking Dashboard

> A professional Business Intelligence dashboard built using Microsoft Power BI to analyze ride booking operations, revenue, cancellations, customer behavior, and service performance.

---

## 📌 Project Overview

This project presents an interactive **Power BI Dashboard** developed using ride booking transaction data. The dashboard transforms raw operational data into clear visual insights for management and business decision-making.

It focuses on key metrics such as:

- Total Bookings
- Completed Rides
- Lost / Cancelled Bookings
- Revenue Performance
- Vehicle Category Trends
- Payment Method Usage
- Driver Ratings
- Booking Trends Over Time

---

## 🎯 Project Objectives

The main objectives of this dashboard are:

- Analyze ride booking performance
- Measure business revenue
- Identify cancellation patterns
- Compare vehicle demand
- Evaluate service quality through ratings
- Monitor daily booking trends
- Improve decision-making using visual analytics

---

## 🛠️ Tools & Technologies Used

| Tool | Purpose |
|------|---------|
| Microsoft Power BI Desktop | Dashboard Development |
| Excel / CSV | Dataset Source |
| Power Query | Data Cleaning & Transformation |
| DAX | KPI Measures & Calculations |
| Data Visualization | Reporting & Insights |

---

## 📂 Dataset Description

The dataset contains booking transaction records with the following fields:

- Customer ID  
- Booking Status  
- Vehicle Type  
- Booking Value  
- Ride Distance  
- Driver Ratings  
- Payment Method  
- Pickup Location  
- Booking Date

---

# 📈 Key Business Insights

- Auto rides contribute major booking volume.
- UPI and Cash are highly preferred payment methods.
- Booking completion rate is strong.
- Lost bookings indicate operational leakage.
- Stable demand with occasional peak spikes.
- Driver ratings help measure service quality.

---

# 💡 Recommendations

- Reduce cancellations through faster driver matching.
- Increase fleet availability during peak hours.
- Promote digital payments with offers.
- Improve driver training for low ratings.
- Expand high-demand vehicle segments.
